# Projeto-PAA---Problema-da-Mochila

<br>

- **Integrantes: Alexander Fábio, João Victor Morais e Gabriel Nascimento.**

<br>

- **Descrição do problema: Dado uma mochila que suporta um peso máximo X e N itens com pesos variados, 
o objetivo é selecionar alguns itens (no máximo 1 de cada) para maximizar o valor total, sem ultrapassar a capacidade da mochila.**

<br>

- **Técnicas utilizadas: Programação Dinâmica e Abordagem Gulosa.**
